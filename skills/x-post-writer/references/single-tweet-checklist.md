# Single-Tweet Checklist

Run every single-tweet draft through this before showing the approval card.

## Fit

- [ ] Under 280 chars on a standard account. Count emoji as 2 chars each.
- [ ] If over 280 and the user is not on X Premium: tighten, or escalate to a
      thread. Never ship a truncated thought.
- [ ] First line stops the scroll on its own (no reliance on a second line).

## Voice scrub

- [ ] No em dashes (`—`), en dashes (`–`), or double dashes (`--`). Use `..`.
- [ ] No AI vocab: leverage, utilize, streamline, harness, delve, foster,
      fundamentally, essentially, game-changer, deep dive.
- [ ] No rule-of-three triplet unless each item is concrete.
- [ ] Names capitalized (Stripe, Claude, Vercel). Lowercase sentence starts are
      fine if the voice calls for it.

## Substance

- [ ] One idea only. Two ideas means two tweets.
- [ ] At least one specific number or named entity where the claim allows it.
- [ ] A clear primary goal: replies, reposts, likes, or bookmarks. Not all four.

## Format

- [ ] 0 or 1 hashtag, at the end.
- [ ] 0 or 1 emoji, and only if it earns its place. None on a serious/contrarian take.
- [ ] No external link in the tweet body (offer to drop it in a reply).
- [ ] Line breaks used as beats, not decoration.

## Close

- [ ] Ends on a landing or a specific question, never "what do you think?".

## Char-count quick reference

| Account | Per-tweet limit | Emoji cost |
|---|---|---|
| Standard | 280 | 2 chars each |
| X Premium | 25,000 | 2 chars each |

Publora reserves ~8 chars per tweet for the `(X/N)` marker when it auto-splits a
long draft into a thread.
